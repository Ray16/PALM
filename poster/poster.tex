% ── MOTIVATION ──
\begin{block}{Motivation}
ML models in drug discovery, materials science, and bioinformatics are often evaluated with \textbf{randomly split} datasets. This introduces \textbf{data leakage}:

\vspace{0.5ex}
\begin{itemize}
  \item Structurally similar entities appear in \textbf{both} train and test sets
  \item Models memorize patterns rather than learning generalizable representations
  \item Reported metrics \textbf{overestimate} real-world performance
\end{itemize}

\vspace{0.8ex}
\begin{center}
\begin{tikzpicture}[
    every node/.style={font=\normalsize},
    box/.style={draw, rounded corners=3pt, minimum width=4.2cm, minimum height=1.4cm, align=center, line width=1.2pt},
]
  \node[box, fill=palmlight, draw=palmblue] (train) {Train Set\\{\scriptsize Drug A, Drug A$'$, Drug B}};
  \node[box, fill=red!8, draw=palmred, right=1.5cm of train] (test) {Test Set\\{\scriptsize Drug A$''$, Drug C}};
  \draw[-{Stealth[length=4mm]}, line width=1.5pt, palmred, dashed]
    (train.east) -- node[above, font=\large\bfseries, palmred] {Leakage!} (test.west);
  \node[below=0.4cm of $(train)!0.5!(test)$, font=\small\color{palmgray}, align=center]
    {A, A$'$, A$''$ share $>$90\% structural similarity};
\end{tikzpicture}
\end{center}

\vspace{0.6ex}
Our experiments show random splitting inflates test R$^2$ by \textbf{0.23--0.33} on molecular property prediction, revealing that apparent performance is largely driven by leakage.

\vspace{0.6ex}
\textbf{Goal:} A unified, automated tool for \textbf{leakage-minimized splitting} across molecules, materials, biomolecules, and genes.
\end{block}

\vspace{0.6ex}
\textbf{DataSAIL} formulates splitting as a constrained optimization problem. \textbf{PALM} extends it with:
\begin{itemize}
  \item Domain-specific \textbf{feature extraction} for 4 entity types
  \item \textbf{Adaptive distance metrics} (Euclidean, cosine, Tanimoto)
  \item \textbf{Scaffold splitting} for chemistry-aware molecule grouping
  \item \textbf{17+ input formats} with automatic detection
  \item Built-in \textbf{quality metrics}, \textbf{visualizations}, and \textbf{ML exports}
  \item Interactive \textbf{web application} with guided tooltips
\end{itemize}
\end{block}

\begin{block}{Key Contributions}
\begin{itemize}
  \item \textbf{Unified framework} for leakage-aware splitting across molecules, biomolecules, materials, and genes
  \item \textbf{Scaffold splitting} groups molecules by Bemis-Murcko core structure --- no scaffold leaks across splits
  \item \textbf{Adaptive distance selection:} Tanimoto for fingerprints, cosine for sparse embeddings, PCA for very high-dimensional data
  \item \textbf{Format preservation:} SDF$\to$SDF, FASTA$\to$FASTA, CIF$\to$CIF --- direct integration into ML pipelines
  \item \textbf{ML exports:} split indices (PyTorch/sklearn) and fold-column CSV per technique
  \item \textbf{Benchmark mode:} automatic comparison across all techniques with recommended best split
  \item \textbf{Quality metrics:} NN leakage, distribution shift, coverage, entity overlap
\end{itemize}
\end{block}

\begin{block}{Limitations \& Future Work}
\begin{itemize}
  \item C2 on very sparse features may yield low coverage --- adaptive relaxation helps but does not eliminate the issue
  \item Scaffold splitting is molecule-specific; analogues for proteins (fold-based) and materials (structure-type) are future work
  \item Scaling to $>$100K entities needs approximate distance methods
  \item Future: cross-validation folds, temporal splits, batch processing
\end{itemize}
\end{block}
